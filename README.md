Ghost Memo is a minimal, single-file web app for sharing encrypted notes that self-destruct. Write a note, choose when it burns (5 minutes, 24 hours, on first read, or manual), and share the generated URL. The decryption key lives only in the URL fragment and is never sent to a server. Optionally add a password for a second layer of protection. All encryption happens in your browser using the Web Crypto API — AES-256-GCM with PBKDF2 key derivation. Inspired by Enclosed and PrivateBin.
